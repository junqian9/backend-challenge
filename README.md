# ExpertFinder

### [Basic info](#basic-info)

- Ruby version: ruby 2.7.2

- Rails version: Rails 6.1.3

- System dependencies: see [Gemfile](./Gemfile)

### [Configuration](#configuration)

1. `bundle install`

2. `rails db:create`

3. `rails db:migrate`

4. `rails db:seed`
